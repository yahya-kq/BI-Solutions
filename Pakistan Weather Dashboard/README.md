# Power BI Live Weather Dashboard for Pakistan

A real-time weather dashboard built with Power BI, delivering live weather insights and forecasts for major cities across Pakistan. The dashboard demonstrates advanced Power BI capabilities including API integration, Power Query transformations, and interactive visualizations.

## Project Overview
This comprehensive Power BI dashboard provides real-time weather conditions, forecasts, and air quality metrics for Pakistan's major cities. It showcases practical applications of business intelligence techniques in real-time data monitoring and visualization.  

**Start Date:** June 22, 2025  
**Purpose:** Real-time weather monitoring and forecasting for decision-making  



## Key Features
- **Live Weather Metrics:** Temperature, humidity, wind speed, UV index  
- **7-Day Forecast:** Extended predictions with sunrise and sunset times  
- **Air Quality Index:** Real-time AQI monitoring with color-coded indicators  
- **Interactive Controls:** City selection and date filtering slicers  
- **Responsive Design:** Clean, user-friendly interface suitable for all devices

- **Data Provider:** Weather API Service (JSON format)  
- **Update Frequency:** Every 15 minutes  
- **Authentication:** Secured API key  



## Dashboard Structure

### Core Components
- **Control Panel:** City selector and date picker slicers  
- **Key Metrics:** Current temperature, weather conditions, humidity cards  
- **Trend Charts:** 7-day temperature trends and forecast visualizations  
- **Air Quality:** AQI monitoring with health indicators  
- **Geographic View:** City comparison and regional context  

### Data Model
- **Fact Tables:** Current weather, forecasts, air quality metrics  
- **Dimension Tables:** Cities, dates, weather conditions lookup  
- **Relationships:** Star schema design for optimal performance
